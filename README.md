## üö¶ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/vaibhav-katkar-dev/Chat-App.git
cd Chat-App
# Make Sure Node JS Installed
npm init -y
npm install express socket.io
node app.js
```

---

Open your browser and go to `http://localhost:3000`.
